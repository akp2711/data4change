# 📁 Your Capstone Website Files

## What's in this folder?

| File | What it is |
|------|------------|
| `index.html` | Homepage — title, stats, key finding, conclusion |
| `graphs.html` | All your charts and graphs with explanations |
| `sources.html` | Datasets, articles, and tools you used |
| `about.html` | About you and how you did the project |

## How to use these files

### Step 1 — Add your graphs
Create a folder inside this folder called `graphs`.
Save all your chart images (as `.png` files) inside it.
Then in `graphs.html`, replace `src="graphs/scatter-plot.png"` with your actual filenames.

### Step 2 — Fill in the ✏️ sections
Open each `.html` file in any text editor and look for lines with a ✏️ comment.
Replace the example text with your real content.

### Step 3 — Upload to GitHub
Upload ALL of these files + your graphs folder to your GitHub repository.

### Step 4 — Enable GitHub Pages
Settings → Pages → Source: main branch → Save.
Your site will be live at: https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/

## Changing the color scheme
Open any file and find the `:root` section near the top.
Change `--accent: #2563eb;` to any color you like.
Use the same color in ALL 4 files to keep it consistent.
Color picker: https://coolors.co
